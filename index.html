Disease Recognition
using OCR & RAG Models

Our intelligent system extracts data from blood reports using Optical Character Recognition (OCR) and analyzes it with a Retrieval-Augmented Generation (RAG) model. It provides AI-driven insights, identifies possible diseases, and assists in early diagnosis.

PROCESS

How the System Works

The workflow combines text extraction, data analysis, and generative reasoning to predict diseases accurately.

Step 1 - Upload Report

Upload your blood report image or PDF. The OCR module extracts essential metrics automatically.

Step 2 - Data Extraction & Retrieval

Extracted terms are compared with medical knowledge bases using RAG to retrieve related medical insights.

Step 3 - Disease Prediction

AI models analyze extracted data to detect possible diseases and provide recommendations or alerts.

About the Project

This project leverages advanced AI techniques to automate disease recognition using OCR and Retrieval-Augmented Generation (RAG). It provides healthcare professionals and patients with intelligent insights from simple blood reports, ensuring faster, data-driven decisions.
